# Product Overview

**Procastify** is a productivity and study management application designed to help users overcome procrastination through structured learning and focus sessions.

## Core Features

- **AI-Powered Summarization**: Uses Google Gemini AI to summarize content from text, YouTube videos, and web articles
- **Smart Note-Taking**: Visual note editor with text blocks, sticky notes, and flashcards
- **Adaptive Routines**: AI-generated study schedules based on user preferences and workload
- **Focus Sessions**: Pomodoro-style focus timer with task tracking
- **Quiz Generation**: Auto-generated quizzes from notes for knowledge retention
- **Progress Tracking**: Study time analytics and login streaks

## User Types

- **Guest Users**: Local-only data storage, no account required
- **Authenticated Users**: Firebase-backed data sync across devices
- **Migration Support**: Seamless transition from guest to authenticated mode

## Key Value Propositions

1. Reduces procrastination through structured, bite-sized tasks
2. Leverages AI to automate content processing and quiz creation
3. Provides guilt-free "Procastify Breaks" to maintain motivation
4. Combines multiple productivity tools in one cohesive interface
